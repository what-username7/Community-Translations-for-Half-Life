#

> [!警告]
> 此仓库是[**`GoldSrc 社区翻译`**](https://github.com/source-br/Community-Translations-for-GoldSrc)的一部分。它设计为作为该项目中央仓库的子模块使用，而中央仓库应当用于管理此仓库。

该项目旨在以社区驱动、玩家对玩家的方式，将**Half-Life**的材质、界面和文本翻译成尚无官方翻译的其他语言。

这些翻译由社区制作，本仓库汇总了可编辑的 `.psd` 格式的材质。文本和界面位于 GoldSrc 社区翻译仓库中。

- [关于翻译](https://github.com/source-br/Community-Translations-for-GoldSrc/wiki/About-the-Translations)

## 可用翻译
- 🇧🇷 葡萄牙语 (巴西) - ⬇️ [下载该翻译](https://github.com/source-br/Community-Translations-for-GoldSrc/releases/download/continuous/Half-Life-Brazilian.zip)

## 如何协作

- [文档](https://github.com/source-br/Community-Translations-for-GoldSrc/wiki)

如果你有相关知识，你可以将游戏翻译成你的语言，或者改进项目。

1. **改进或修正建议**
- 在此仓库中打开一个 *issue*。
- 或通过 [Discord](https://discord.gg/tVNv6SNZZT) 联系我们。

2. **其他语言的翻译或改进**
- 阅读文档，如果需要进一步指导，请联系团队。
- 经验证后，你的翻译将被添加到项目中，并获得应有的署名。
